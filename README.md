# Real Estate Website Demo – Personal Project

This is a small personal project built to experiment with HTML, CSS, and basic front-end structure.

The idea was to prototype a website for a fictional family-owned real estate and property management company, focusing on clear messaging, trust, and structured service presentation.

## 🌐 Live Demo

https://Marco-Nanni99.github.io/real-estate-website-demo/

## 🎯 Purpose

- Practice writing clean semantic HTML
- Structure a multi-section landing page
- Experiment with layout and styling
- Test basic UI interactions
- Learn how to deploy using GitHub Pages

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Git & GitHub
- GitHub Pages
